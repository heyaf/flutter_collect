# flutter_collect
flutter项目—登录注册模块
第一个flutter实战项目-登录注册模块
包含了登录、注册和找回密码三个界面实现，具体效果见下图。

### 几个技术点：
**1. 输入框的监听**
**2. 手机号的正则验证**
**3. androidID获取设备唯一标识**
**4. 建立网络请求**
